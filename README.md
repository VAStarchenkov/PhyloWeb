# PhyloWeb

Веб-интерфейс для построения конфигурации модели Aspartik с поддержкой состояний, приоров, операторов и параметров MCMC.

Приложение позволяет в интерактивном режиме собирать конфигурацию,
проверять корректность данных и экспортировать результат в формат JSON.

---

## Способы взаимодействия

- Настройка состояний (States)
- Конфигурация приоров (Priors)
- Управление операторами (Operators)
- Настройка likelihood и MCMC
- Валидация параметров
- Просмотр итогового JSON
- Экспорт конфигурации

---

## Запуск

Необходима установка Node.js (версии 18+ рекомендуется).

```bash
git clone https://github.com/VAStarchenkov/PhyloWeb.git
cd aspartik-ui
npm install
npm run dev
```

После запуска приложение будет доступно по вашему локальному адресу (в терминале будет видно), например:
```
http://localhost:5173
```

---

## Формат получаемой конфигурации

Итоговая конфигурация формируется в формате JSON и включает:

* states
* priors
* operators
* likelihood
* mcmc
